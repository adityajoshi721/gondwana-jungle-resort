# Gondwana Jungle Resort - React Website

Welcome to the **Gondwana Jungle Resort** website repository! This project is a premium website built using **React** to showcase the resort’s offerings, amenities, gallery, and contact information with a clean and elegant design.

## 🌿 Live Demo

🔗 [**gondwanajungleresort.com**](https://gondwanajungleresort.com) *(Once Deployed)*

---

## 🏗️ Tech Stack

- **Frontend:** React, Tailwind CSS
- **Backend (if applicable):** Spring Boot, PostgreSQL
- **Hosting:** Vercel / Netlify / Hostinger
- **Version Control:** Git & GitHub

---

## 📥 Installation & Setup

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/adityajoshi721/gondwana-jungle-resort.git
cd gondwana-jungle-resort
```

### 2️⃣ Install Dependencies

```bash
npm install
```

### 3️⃣ Start the Development Server

```bash
npm start
```

- The website will be available at [**http://localhost:3000/**](http://localhost:3000/).

---

## 🚀 Deployment Guide

### **1️⃣ Deploy on Vercel (Recommended)**

1. Push your code to GitHub.
2. Go to [**Vercel**](https://vercel.com/) and log in.
3. Import the repository and deploy.
4. Connect your domain `gondwanajungleresort.com`.

### **2️⃣ Deploy on Netlify**

1. Push your code to GitHub.
2. Go to [**Netlify**](https://www.netlify.com/) and log in.
3. Import the repository and deploy.

### **3️⃣ Deploy on Hostinger (cPanel)**

1. Build the project:
   ```bash
   npm run build
   ```
2. Upload the `build/` folder to `public_html` via **cPanel** or **FTP**.
